1. **Click Deploy Button**
   ```
   https://render.com/deploy?repo=https://github.com/Nisu7648/R3SN
   ```

2. **Setup MongoDB Atlas**
   - Create free cluster
   - Get connection string
   - Whitelist IPs (0.0.0.0/0)

3. **Configure Render**
   - Set `MONGODB_URI`
   - Other vars auto-generate
   - Click "Apply"

4. **Wait & Test**
   - Wait 5-10 minutes
   - Test: `curl https://your-app.onrender.com/health`
   - Done!
